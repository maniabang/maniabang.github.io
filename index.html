✈️ Travel Record Flutter App

여행 기록 & 항공권 검색 Flutter 앱. 출입국 증명서 OCR로 국가/날짜를 자동 추출하고, 여행 기록을 저장한 후 항공권 최저가 검색까지 한 번에 처리하는 실사용 가능한 여행 플랫폼

🛠️ 핵심 기술 스택

Flutter google_ml_kit Riverpod Hive Dio OCR PWA
  • Google ML Kit OCR 로컬 처리 - API 키 불필요, 무료 한글 인식으로 출입국 증명서 텍스트 추출
  • file_picker 다중 파일 업로드 - PDF/이미지 동시 지원으로 사용자 편의성 극대화
  • Riverpod 상태 관리 - Provider 패턴으로 경량화된 앱 상태 관리
  • Hive 로컬 데이터베이스 - NoSQL 기반 오프라인 여행 기록 저장/조회
  • Dio HTTP 클라이언트 - 항공권 API 연동 및 최저가 검색 기능
  • Regex 파싱 최적화 - OCR 결과에서 국가명/날짜 정확도 개선
